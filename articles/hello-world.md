# Hello, world

This is the first thing I've written here. It's also a quick tour of what this little
writing space can do, so I remember the formatting later.

## Why I'm writing

I like to think on paper. Putting an idea into words is the fastest way to find out
whether I actually understand it. So this is where some of that will live.

## The formatting works

You get the usual Markdown:

- **Bold** and *italic* text
- Lists, like this one
- Links, like [Querio](https://querio.ai)

> And blockquotes, for the lines worth setting apart.

Even code, inline like `const x = 42` and in blocks:

```js
function hello(name) {
  return `Hello, ${name}`;
}
```
